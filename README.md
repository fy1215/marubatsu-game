# マルバツゲーム

JavaScriptで作った最初のゲームプロジェクト。
HTMLとCSSはClaudeに作ってもらいました。

## 機能
- 3×3グリッド
- プレイヤー交互切り替え（O と X）
- 勝敗判定（8パターン）
- 引き分け判定
- リセット機能

## 使った技術
- HTML (AI)
- CSS  (AI)
- JavaScript

## 学んだこと
- DOM操作（getElementById、querySelector）
- イベントリスナー（addEventListener）
- 配列操作（forEach、some、includes）
- 条件分岐とゲームロジック
- 状態管理（gameActive変数）
